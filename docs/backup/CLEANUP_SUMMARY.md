## 🚀 Доступные команды

Все инструменты мониторинга теперь доступны из папки `/monitoring/`:

```bash
cd /root/vpnbot/monitoring

# Мониторинг логов в реальном времени
./monitor-logs
python3 log_monitor.py --journal --follow

# Управление ботом
./manage-bot status
./manage-bot restart

# Управление VPN
./vpn-control.sh status

# Просмотр инструментов
./show-log-tools
```
